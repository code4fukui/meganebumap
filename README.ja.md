# メガネブマップ
アニメ「メガネブ！」の聖地巡礼を支援するためのWebアプリケーションです。

## デモ
http://codeforfukui.github.io/meganebumap/

## 機能
- アニメのオープニングで登場した舞台となった場所を地図上に表示
- 各スポットの詳細情報（写真、説明、URL）を表示
- ユーザーの現在地を表示し、周辺の聖地を確認できる

## 必要環境
特に必要なものはありません。Web ブラウザがあれば利用できます。

## 使い方
ウェブサイトにアクセスして地図上のアイコンをクリックすると、各スポットの詳細が表示されます。
位置情報の取得を許可すると、現在地周辺の聖地も表示されます。

## データ・API
アニメの舞台となった場所の情報は、Linkdata.orgのオープンデータを利用しています。
Google Maps APIを使用して地図表示を行っています。
